# FraudShield: Detecting Credit Card Fraud with ML

## Abstract
This project focuses on detecting fraudulent credit card transactions using machine learning. The dataset contains real-world transaction details and is highly imbalanced. SMOTE is used to handle imbalance, and a Random Forest model is used for prediction.

## Problem Statement
Detect fraudulent transactions from highly imbalanced financial data using machine learning techniques.

## Dataset
- Source: Kaggle
- Records: ~1.29 million
- Features: transaction amount, merchant, category, location

## Methodology
- Data preprocessing
- Feature engineering
- Encoding categorical data
- SMOTE for imbalance
- Random Forest model

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Seaborn, Matplotlib

## Results
- High recall (82%) for fraud detection
- Effective model performance using Random Forest

## Conclusion
The model successfully detects fraudulent transactions and can be extended for real-time applications.
